Clé API OpenRouter Manquante

La fonctionnalité {{ service|title }} nécessite une clé API OpenRouter pour fonctionner. Cette clé permet d'utiliser l'intelligence artificielle pour extraire les données des PDFs.

Comment configurer la clé API

  1. Obtenez une clé API gratuite sur openrouter.ai
  2. Dans Replit, ouvrez l'onglet "Secrets" (icône de cadenas)
  3. Ajoutez un nouveau secret:
    • Clé: OPENROUTER_API_KEY
    • Valeur: Votre clé API OpenRouter
  4. Redémarrez l'application
