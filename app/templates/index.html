Comment ça marche ?

Télécharger & Zipper

  1. Entrez vos URLs de PDFs (une par ligne)
  2. Cliquez sur "Télécharger et Zipper"
  3. Attendez le traitement
  4. Téléchargez votre ZIP

Fusionner PDFs

  1. Sélectionnez votre fichier ZIP
  2. Cliquez sur "Fusionner"
  3. Attendez le traitement
  4. Téléchargez le PDF fusionné et l'Excel

Analyse Intelligente

  1. Choisissez votre type d'input (CSV, ZIP ou PDF)
  2. Uploadez votre fichier
  3. L'IA analyse et extrait les données
  4. Téléchargez le fichier Excel généré
